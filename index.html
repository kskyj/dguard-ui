D-Guard 페이지 개선 항목

이 전달물은 화면과 인터랙션 설명을 위한 정적 HTML/CSS/JS 목업입니다.

Page 01 점검대상

등록된 모든 DB를 테이블로 조회하고, DB명/라벨 검색, 상태 필터, 라벨 일괄 수정, 선택 DB 점검 실행, Excel 내보내기를 확인하는 메인 페이지입니다.

 Page 02 검출목록

선택 DB의 검출 경로, 개인정보 내용, 상태변경, 이행점검을 확인하는 상세 페이지입니다.

 Page 03 점검이력

동일한 D-Guard 셸을 유지하면서 분석 실행 이력, 상태, 담당자, 검출건수 비교를 보여주는 목록 중심형 페이지입니다.

 Page 04 제외신청관리

검출단위 기준 제외신청 접수 현황과 승인·거부 처리 결과를 탭으로 분리해 관리하는 페이지입니다.

 Page 05 조치계획관리

조치필요 검출단위를 기반으로 조치계획서 작성, 업로드, 이행점검 결과를 관리하는 페이지입니다.
