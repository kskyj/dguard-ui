업체 전달용 정적 목업

D-Guard 페이지 목차

이 전달물은 서버 연동용이 아니라 화면 구조와 인터랙션 설명을 위한 정적 HTML/CSS/JS 목업입니다. 아래 페이지를 개별 파일로 열어 검토하면 됩니다.

Page 01 점검대상

등록된 모든 DB를 테이블로 조회하고, DB명/라벨 검색, 상태 필터, 라벨 일괄 수정, 선택 DB 점검 실행, Excel 내보내기를 확인하는 메인 페이지입니다.

 Page 02 검출목록

선택 DB의 검출 경로, 상태 편집, 개인정보 하위 목록, 이행점검 흐름을 확인하는 상세 목업 페이지입니다.

 Page 03 점검이력

동일한 D-Guard 셸을 유지하면서 분석 실행 이력, 상태, 담당자, 검출건수 비교를 보여주는 목록 중심형 페이지입니다.
