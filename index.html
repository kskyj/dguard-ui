D-Guard 페이지 개선 항목

이 전달물은 화면과 인터랙션 설명을 위한 정적 HTML/CSS/JS 목업입니다. 각 화면의 설명은 핵심 내용입니다.

Page 01 검출목록

각 검출 경로별로 최근 점검내역을 확인합니다. 검출 경로를 확인하고, 검출된 개인정보 내용을 확인하고, 상태를 관리하며 빠르게 이행점검을 할 수 있습니다.

 Page 02 점검이력

점검대상 DB/테이블/컬럼 단위와 검출룰, 반복주기 등으로 점검스케쥴을 생성하며, 점검이력은 삭제가 가능하고, 재사용하기 쉽게 제공합니다.

 Page 03 제외신청관리

검출단위 기준 제외신청 접수 현황과 승인·거부 처리 결과를 탭으로 분리해 관리하는 페이지로

 Page 04 조치계획관리

즉지 조치가 불가능한 검출건에 대해서, 검출 경로, 조치방법, 조치예정일을 포함한 조치계획서 작성하고, 출력물을 업로드 합니다. 검출 경로를 바로 이행점검하여 조치결과를 확인 할수 있습니다.

 Page 05 점검대상

등록된 모든 DB 기준으로 각DB의 정보(점검대상 정보), 최근 검출 결과(검출목록)를 제공하며, 부서명/업무명 등은 그룹은 라벨을 통해 관리합니다.

 Page 06 점검대상 상세

점검 대상의 DB의 정보, 테이블 별 담당자, 변경정보 관리등과 해당 DB의 점검 이력(최근 1년)을 제공합니다.
